Group Theory

1 Basic Mathematical Background

1.3 Basic Definitions

Definition 1. The order of a group \(\equiv\) the number of elements in the group.

Definition 2. A subgroup \(\equiv\) a collection of elements within a group that by themselves form a group.

Theorem. If in a finite group, an element X is multiplied by itself enough times (n), the identity \(X^n = E\) is eventually recovered.

Definition 3 The order of an element \(\equiv\) the smallest value of n in the relation \(X^n = E\).

Definition 4. The period of an element X ≡ collection of elements \(E, X, X^2, . . . , X^{n-1}\), where n is the order of the element. The period forms an Abelian subgroup.

1.4 Rearrangement Theorem

If \(E,A_1,A_2, . . . , A_h\) are the elements of

a group, and if \(A_k\) is an arbitrary group element, then the assembly of elements \(A_kE,A_kA_1, . . .,A_kA_h\) contains each element of the group once and only once.

1.5 Cosets

The cosets \(\to\) factor group \(\to\) space groups

Definition 5. If B is a subgroup of the group G, and X is an element of G, then the assembly \(EX,B_1X,B_2X, . . .,B_gX\) is the right coset of B, where B consists of \(E,B_1,B_2, . . .,B_g\).

A coset need not be a subgroup.

Theorem. Two right cosets of given subgroup either contain exactly the same elements, or else have no elements in common.

Theorem. The order of a subgroup is a divisor of the order of the group.

### 1.6 Conjugation and Class

Definition 6. An element B conjugate to A is by definition \(B \equiv XAX^{-1}\), where X is an arbitrary element of the group.

Theorem. If B is conjugate to A and C is conjugate to B, then C is conjugate to A.

Definition 7. A class is the totality of elements which can be obtained from a given group element by conjugation.

Note that each class corresponds to a physically distinct kind of symmetry operation.

Theorem. All elements of the same class have the same order.

Definition 8. A subgroup B is self-conjugate (or invariant, or normal ) if \(XBX^{-1}\) is identical with B for all possible choices of X in the group.

To form a self-conjugate subgroup, it is necessary to include entire classes in this subgroup.

Definition 9. A group with no self-conjugate subgroups \(\equiv\) a simple group.

Theorem. The right and left cosets of a self-conjugate subgroup N are the same.

\[XN_i=XN_iX^{-1}X=N_jX\]

Theorem. The multiplication of the elements of two right cosets of a selfconjugate subgroup gives another right coset.

\[(NX)(NY) = N_iXN_lY=N_i{XN_l}Y=N_i(N_mX)Y=(N_iN_m)(XY)=N(XY)\]

1.7 Factor Groups

Definition 10. The factor group (or quotient group) is constructed with respect to a self-conjugate subgroup as the collection of cosets of the selfconjugate subgroup, each coset being considered an element of the factor group.

Definition 11. The index of a subgroup \(\equiv\) total number of cosets = \((order of group)/ (order of subgroup)\).

The order of the factor group is the index of the self-conjugate subgroup.

1.8 Group Theory and Quantum Mechanics

The group of symmetry operators \(\hat{P}_R\) which leave the Hamiltonian invariant.

\(\hat{P}_R\) are symmetry operations of the system and the \(\hat{P}_R\) operators commute with the Hamiltonian.

\(\hat{P}_R\) \(\to\) the group of the Schrodinger equation

2 Representation Theory and Basic Theorems

2.1 Important Definitions

Definition 12. Two groups are isomorphic or homomorphic if there exists a correspondence between their elements such that \[ A\to\hat{A}\\\\ B\to\hat{B}\\\\ AB\to\hat{A}\hat{B} \] Definition 13. A representation of an abstract group is a substitution group (matrix group with square matrices) such that the substitution group is homomorphic (or isomorphic) to the abstract group. We assign a matrix \(D(A)\) to each element A of the abstract group such that $ D(AB) = D(A)D(B)$.

First of all, we will find that an eigenfunction for a quantum mechanical operator will transform under a symmetry operation similar to the application of the matrix representing the symmetry operation on the matrix for the wave function.

Secondly, quantum mechanical operators are usually written in terms of a matrix representation, and thus it is convenient to write symmetry operations using the same kind of matrix representation.

Finally, matrix algebra is often easier to manipulate than geometrical symmetry operations.

2.2 Matrices

Definition 14. Hermitian matrices are defined by: \(\widetilde{A}=A^*,\widetilde{A}^\ast=A, A^\dagger=A\)

Unitary matrices are defined by: \(\widetilde{A}^*=A^\dagger=A^{-1}\);

Orthonormal matrices are defined by: \(\widetilde{A}=A^{-1}\).

Definition 15. The dimensionality of a representation is equal to the dimensionality of each of its matrices, which is in turn equal to the number of rows or columns of the matrix.

These representations are not unique.

To overcome the difficulty of non-uniqueness of a representation with regard to a similarity transformation, we often just deal with the traces of the matrices which are invariant under similarity transformations. The trace of a matrix is defined as the sum of the diagonal matrix elements.

To overcome the difficulty of the ambiguity of representations in general, we introduce the concept of irreducible representations.

2.3 Irreducible Representations

If by one and the same equivalence transformation, all the matrices in the representation of a group can be made to acquire the same block form, then the representation is said to be reducible; otherwise it is irreducible. Thus, an irreducible representation cannot be expressed in terms of representations of lower dimensionality.

\(\Gamma_R\) \[ \begin{pmatrix} \Gamma_1&O&O\\\\O&\Gamma_{1'}&O\\\\O&O&\Gamma_2 \end{pmatrix} \] \[\Gamma_R=\Gamma_1+\Gamma_{1'}+\Gamma_2\]

Assume \(\Gamma_R\) is a reducible representation for some group \(G\) but an irreducible representation for some other group \(G''\).

when a perturbation is applied to lower the symmetry, the degeneracy of the energy levels tends to be reduced.

2.4 The Unitarity of Representations

Theorem. Every representation with matrices having nonvanishing determinants can be brought into unitary form by an equivalence (similarity) transformation.

\[\hat{\hat{A}_x}=d^{-1/2}U^{-1}A_xUd^{1/2}\]

\[H=\sum_{x=1}^hA_xA_x^{\dagger}\]

\[d=\sum_{x=1}^h\hat{A}_x\hat{A}_x^{\dagger}\]

\(U\) is the unitary matrix that diagonalizes the Hermitian matrix \(H\) and \(\hat{A}_x=U^{-1}A_xU\).

2.5 Schur's Lemma part1

Lemma. A matrix which commutes with all matrices of an irreducible representation is a constant matrix, i.e., a constant times the unit matrix. Therefore, if a non-constant commuting matrix exists, the representation is reducible; if none exists, the representation is irreducible.

2.6 Schur's Lemma part2

Lemma. If the matrix representations \(D^{(1)}(A_1),D^{(1)}(A_2), . . . , D^{(1)}(A_h)\) and \(D^{(2)}(A_1),D^{(2)}(A_2), . . . , D^{(2)}(A_h)\) are two irreducible representations of a given group of dimensionality \(\ell_1\) and \(\ell_2\), respectively, then, if there is a matrix of \(\ell_1\) columns and \(\ell_2\) rows M such that

\[MD^{(1)}(A_x) = D^{(2)}(A_x)M\]

for all \(A_x\), then \(M\) must be the null matrix \((M=O)\) if \(\ell_1 \neq \ell_2\). If \(\ell_1 = \ell_2\), then either \(M=O\) or the representations \(D^{(1)}(A_x)\) and \(D^{(2)}(A_x)\) differ from each other by an equivalence (or similarity) transformation.

3 Character of a Representation

if \(D^{(\Gamma_j)}(R)\) is a representation of a group, so is \(U^{−1}D^{(\Gamma_j)}(R)U\). To get around this arbitrariness, we introduce the use of the trace (or character) of a matrix representation which remains invariant under a similarity transformation.

3.1 Definition of Character

Definition 17. The character of the matrix representation \(\chi^{(\Gamma_j)}(R)\) for a symmetry operation \(R\) in a representation \(D^{(\Gamma_j)}(R)\) is the trace (or the sum over

diagonal matrix elements) of the matrix of the representation:

\[\chi^{(\Gamma_j)}(R)=traceD^{(\Gamma_j)}(R)=\sum_{\mu=1}^{\ell_j}D^{(\Gamma_j)}(R)_{\mu\mu}\]

where \(\ell_j\) is the dimensionality of the representation \(\Gamma_j\) and \(j\) is a representation

index.

From the definition, it follows that representation \(\Gamma_j\) will have \(h\) characters, one for each element in the group.

3.2 Characters and Class

Theorem. The character for each element in a class is the same.

3.3 Wonderful Orthogonality Theorem for Character

Theorem. The primitive characters of an irreducible representation obey the orthogonality relation

\[\sum_R\chi^{(\Gamma_j)}(R)\chi^{(\Gamma_{j'})}(R^{-1})=h\delta_{\Gamma_j,\Gamma_{j'}}\]

\[\sum_R\chi^{(\Gamma_j)}(R)\chi^{(\Gamma_{j'})}(R)^*=h\delta_{\Gamma_j,\Gamma_{j'}}\]

where \(\Gamma_j\) denotes irreducible representation \(j\) with dimensionality \(\ell_j\).

Theorem. A necessary and sufficient condition that two irreducible representations be equivalent is that the characters be the same.

3.4 Reducible Representations

Theorem. The reduction of any reducible representation into its irreducible constituents is unique.

if \(\chi(C_k)\) is the character for some class in a reducible representation

\[\chi(C_k)=\sum_{\Gamma_i}a_i\chi^{(\Gamma_i)}(C_k)\]

\[\sum_kN_k[\chi^{(\Gamma_j)}(C_k)]^*\chi(C_k)=S_j\]

\[a_j=\frac{S_j}{h}\]

3.5 The Number of Irreducible Representations

3.7 Regular Representation

Theorem. The regular representation contains each irreducible representation a number of times equal to the dimensionality of the representation.
